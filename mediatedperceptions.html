Mediated Perceptions

Mediated Perceptions is a project exploring alternative sensory experiences made possible with augmented reality. I built two cameras that attached to the front of an oculus rift that mimmicked the field of view of the human eye. After that I built customised C++ software that in realtime manipulated the images before they were presented to the viewer. I built the software so that it would manipulate these images in response to sound in the environment and to music.

This project was featured in TechCrunch, Prosthetic Knowledge, Golem and virtualrealityreviewer.
