Converse

These are styleframes I created for FutureDeluxe's commision by Converse for a new adverstising campaign, which went on to be this great campaign. Building upon my experiments with slitscanning I build custom C++ code software to slice up, randomize and combine multiple images and videos, I even went as far as building a version with a GUI so it could be used by other designers in the studio when I had left.
