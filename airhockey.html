Air Hockey Robot and Score Counter

Two projects carried out for the Postgraduate Physical Computing course I took as part of my Msci in Creative Computing at Goldsmiths. The first project was a score counter using laser tripwires to determine if a point had been scored, and a four digit, seven segment display to count and display the scores. The second project was a robot to play air hockey, adapted using the parts of a RepRap 3D printer. It was based on this open source project by JJRobots. I made a glove controller to control the robot using a flex sensor.

You can read more about the projects on my physical computing project blog.
